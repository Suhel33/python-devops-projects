# Project - Internal DevOps Utilities API

## Aim

This Project aims to deliver Production grade Utilities APIs to internal teams
- AWS Resiurces APIs
- Metrics
- Log Analysis

## Usage 

'git clone<>'

### SetUp python environment
python3.14 -m venv p2env
p2env/Scripts/Activate.ps1  -- for Windows cmd
source p2env/bin/activate   -- for MacOS

### install requirements
pip install -r requirements.txt

### run application
python main.py
